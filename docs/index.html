promptastic

Mechanistic interpretability pipeline for transformer models. Causal attribution, per-head decomposition, MLP analysis, feature decomposition, and more — all from a single forward pass.

Seven capabilities in one pipeline

Each can be enabled independently via CLI flags or combined in a single run.

1 Activation Patching

Zero or mean-ablate a region's contribution at any layer. Measures KL divergence and logit diff to establish causal importance — not just correlation.

--patching

2 Per-Head Attention

Full per-head attention decomposition. Identifies specialist heads, computes entropy, and reveals which heads focus on which prompt regions.

--per-head

3 MLP Capture

Hooks MLP sublayers to capture what the MLP adds or overwrites at each layer. Projects MLP deltas through logit lens to see promoted concepts.

--mlp

4 Tuned Lens

Per-layer affine probes (Belrose et al. 2023) for cleaner intermediate predictions. Train probes on a calibration set, then use as a drop-in logit lens replacement.

--tuned-lens PATH

5 SAE Decomposition

Sparse autoencoders decompose the residual stream into interpretable features. See which concepts activate in which prompt regions at each layer.

--sae PATH

6 Gradient Attribution

Vanilla and integrated gradient methods. Answers "how much does perturbing this region change the output?" — complementary to attention.

--gradients

7 Cross-Step Tracking

Tracks attention dynamics across autoregressive generation steps. Reveals how prompt region influence evolves as the model generates tokens.

--generation

Four-stage pipeline

Stage 1

Prep

Annotate regions in your prompt via markers, regex, or char ranges

Stage 2

Engine

Forward pass with composable hooks capturing all requested modes

Stage 3

Render

11 renderers: heatmaps, cooking curves, GIFs, patching grids, dashboards

Stage 4

Analysis

N-variant comparison, markdown reports, causal scoring, head analysis

Usage

Prepare inputs
python -m promptastic.prep.inputs \
    --prompt system_prompt.txt \
    --regions regions.json \
    --conversations conversations.json \
    --output test_cases.json
Run analysis (all capabilities)
promptastic --input test_cases.json \
    --output results/ \
    --model-path /path/to/model \
    --per-head --mlp --patching --gradients
Render visualizations
# Attention heatmap
python -m promptastic.render.heatmap --result results/case_0.json

# Cooking curves (per-region trajectories)
python -m promptastic.render.cooking_curves --result results/case_0.json

# Activation patching grid
python -m promptastic.render.patching_heatmap --result results/case_0.json

# Per-head attention grid
python -m promptastic.render.head_grid --result results/case_0.json

# SAE feature dashboard
python -m promptastic.render.feature_dashboard --result results/case_0.json
Compare variants
python -m promptastic.analysis.compare \
    --base-dir results/ \
    --variants baseline:Baseline modified:Modified \
    --metrics all,patching,gradient
